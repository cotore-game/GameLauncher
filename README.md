# GameLauncher
分散したゲームビルドを一括管理・配信するためのデスクトップランチャーです。

## About
複数のPCゲームプロジェクトのデプロイとバージョン管理を簡略化するために設計されたハブアプリケーションです。 
開発者がビルドをアップロードし、ユーザーが最小限の手間で最新版をプレイできる環境を提供する目的で制作しています。

## Core Features
登録されたゲームタイトルのカタログ表示と管理。
リモートストレージからの自動ダウンロードおよび展開。
適切な引数を用いた実行ファイルの起動管理。
Windowsスタートメニューへのショートカット登録機能。
GitHub Actionsによるインストーラーの自動生成。

## For Developers
このリポジトリは GitHub Actions を利用した自動ビルドを前提としています。 main ブランチへのプッシュにより、Unityのビルドからインストーラー（.exe）の生成までが自動的に実行されます。

UNITY_LICENSE 等のシークレットをGitHubに登録。
プッシュ後、GitHub Actionsの Actions タブから成果物を取得。

## Game Distribution
配信するゲームデータは、以下の構造でサーバー（GitHub Releases / Cloudflare R2等）に配置することを前提に作成しました。

```
Game_v1.0.zip (ゲーム本体)
manifest.json (バージョン管理用メタデータ)
```
